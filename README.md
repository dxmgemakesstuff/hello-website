# Hello Website

Welcome to the "Hello" project! This website is designed to provide a warm welcome and information about TTC (The Trading Community). 

## Project Overview

The "Hello" website is a simple yet beautiful landing page that aims to greet visitors and provide them with essential information about TTC. 

## File Structure

- `index.html`: The main HTML document that structures the webpage.
- `scripts/main.js`: JavaScript file that adds interactivity to the website.
- `styles/main.css`: CSS file that styles the website for a visually appealing experience.
- `README.md`: Documentation for the project.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the website.
3. Ensure that all files are in the same directory for proper linking.
